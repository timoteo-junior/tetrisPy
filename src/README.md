### **Informações do jogo**

O projeto é um Tetris criado totalmente em Pygame, seguindo o tutorial do [Programming with Nick](https://www.youtube.com/watch?v=nF_crEtmpBo).
Nele aprendemos o uso da Programação Orientada a objetos, manipulando elementos dentro do código e criando módulos. 

No jogo é possível pontuar de acordo com as linhas completas e descendo a peça.
* Descer a peça: 1 ponto
* 1 linha: 100 pontos
* 2 linhas: 300 pontos
* 3 linhas: 500 pontos
* 4 linhas: 1000 pontos

### **Controles:**

* ↑ - Gira a peça
* ← - Move a peça para a esquerda
* → - Move a peça para a direita
* ↓ - Desce a peça

### **Requisitos:**
* Ter o python 3.X instalado na máquina
* Ter o Pygame instalado na máquina
* Se não tiver o Pygame, basta digitar `pip install pygame` no cmd, tendo o python instalado
* Windows 10 ou 11
